# ChessOpeningTrainer_Unity

> **Note:** This project is currently **In Work** (WIP). 
> Currently, it has only been tested on the Samsung s10e Simulator.

## About The Project
A Chess Application built with Unity, designed to help players memorize and practice own created chess openings, analyze games, and train endgame positions against the stockfish engine.
Furthermore there are modes, in which you can improve your chessboard-vision, memorizing patterns and expand your general Knowlege about the Game.

## Prerequisites
Before you begin, ensure you have met the following requirements:
* **Unity:** Version 6000.2.7f2

## Installation & Setup
To run this project locally, you need to clone the repository and manually add the Stockfish engine.
